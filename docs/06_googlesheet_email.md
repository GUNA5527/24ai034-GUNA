- Step 1: Open n8n and create a new workflow.
- Step 2: Add a Google Sheets Trigger node and set it to trigger when a new row is added.
- Step 3: Add a Gmail – Send a Message node.
- Step 4: Connect Google Sheets Trigger → Gmail Send a Message.
- Step 5: Enter the email details (To, Subject, Message), then save and activate the workflow.
